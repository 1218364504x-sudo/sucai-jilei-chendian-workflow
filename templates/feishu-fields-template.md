# 飞书字段模板

## 日常评级字段

| 字段名 | 类型 | 是否日常显示 |
|---|---|---|
| 标题 | 文本 | 是 |
| 故事简介 | 多行文本 | 是 |
| 拍摄可行性 | 单选 | 是 |
| 星级评级 | 单选 | 是 |

## 同步和排查字段

| 字段名 | 类型 | 说明 |
|---|---|---|
| 文件路径 | 文本 | 同步主键 |
| Obsidian链接 | URL | 打开本地故事卡 |
| 日期 | 日期/文本 | 故事卡日期 |
| 分类 | 单选/文本 | 一级分类 |
| 国家地区 | 文本 | 发生地 |
| 来源类型 | 文本 | 新闻、书籍、论文、视频等 |
| 来源链接 | URL | 原始来源 |
| 来源名称 | 文本 | 来源标题 |
| 来源语言 | 文本 | 中文、英语等 |
| 事件时间 | 文本 | 真实事件发生时间 |
| 标签 | 文本 | 主题标签 |
| 改编潜力 | 数字 | AI 或人工判断 |
| 适合形式 | 文本 | 短片、长片、纪录片等 |
| 开发优先级 | 单选/文本 | 星级和拍摄可行性组合后的开发去向 |
| 低成本切口 | 多行文本 | 高价值但难拍故事的可执行入口 |
| 来源可信度 | 单选/文本 | 高、中、低、待核验 |
| AI建议星级 | 单选 | AI 参考 |
| AI建议理由 | 多行文本 | AI 参考理由 |
| 知识库收纳状态 | 单选/文本 | 是否入库 |
| 最后修改时间 | 日期/文本 | 同步排查 |

## 单选值

`星级评级`：

- 1星
- 2星
- 3星
- 4星
- 5星
- 6星

`来源可信度`：

- 高
- 中
- 低
- 待核验

`拍摄可行性`：

- A_低成本可拍
- B_中等成本
- C_高难度
- D_暂不适合直接拍

`开发优先级`：

- 待补拍摄判断
- 优先开发
- 高价值储备
- 低成本练手
- 暂存
- 删除候选
