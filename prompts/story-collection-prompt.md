# 故事收集提示词

```text
请帮我寻找适合导演开发的真实故事素材。

筛选标准：
1. 先去包装：剔除标题党、宣传话术、营销口号、媒体情绪和夸张形容词。
2. 先拆事实：区分已发生事实、当事人说法、机构说法、媒体判断和待核验信息。
3. 不要搬运宣传报道、通稿、表彰稿、品牌故事和热点新闻；除非能挖出具体过程、矛盾、失败、代价、人物关系和可拍细节。
4. 必须有清楚的改编拍摄点：人物/群体/地点/物件、时间链条、动作、空间、道具、声音、关系或处境变化。
5. 必须有可核验来源。
6. 不要只找热点，要找能拍成电影、短片、微电影、纪录片或人物原型的材料。
7. 一句话噱头不是可拍性。类似“因为蓝牙名飞机返航”的新闻钩子，如果展开后只有等待、通报、检查和解释，就跳过。
8. 当前优先微电影、短片和小切口人物故事；每条候选都要能回答 5-30 分钟里人物、关系或处境怎样推进。
9. 不主动收集战争、军事、特工、情报网络、战争英雄、种族制度冲突、族裔对抗、种族隔离和民权斗争类素材。
10. 正式入库前过四道门：人物门、关系门、时间门、影像门。

输出要求：
1. 标题必须像一个可拍瞬间，从动作、声音、道具、关系代价或结尾余味里提取。
2. 故事简介必须是一段完整故事，包含人物、时间、地点、发生了什么、后来怎样、具体结果。
3. 国外/非中国素材简介必须走 `stop-slop -> humanizer -> DeepSeek Pro -> shuorenhua`：先用两个英文去 AI 味 skill 清理事实资料，再由 DeepSeek Pro 重写中文故事简介，最后用 `shuorenhua` 做中文验收。中国素材默认走 `DeepSeek Pro -> shuorenhua`。
4. DeepSeek 重写必须留下可查询审计记录；没有审计记录，不要声称已经由 DeepSeek 重写。翻译腔、评论式末句和词表命中只作为质量提醒，不作为机械拦截。
5. 给出来源链接。
6. 给出 AI建议星级 和 AI建议理由。
7. 对每条候选说明“入库理由”或“跳过原因”。
8. 不要填写正式星级，正式星级由我在飞书确认。
```
