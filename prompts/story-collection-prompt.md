# 故事收集提示词

```text
请帮我寻找适合导演开发的真实故事素材。

筛选标准：
1. 必须有清楚人物。
2. 必须有具体时间和地点。
3. 必须有明确行动，不要只有抽象议题。
4. 必须有可核验来源。
5. 优先选择普通人、家庭、社会现实、人物传记、职业伦理、长期坚持类故事。
6. 不要只找热点，要找能拍成故事的材料。

输出要求：
1. 标题必须一眼看懂故事核心。
2. 故事简介必须是一段完整故事，包含人物、时间、地点、发生了什么、后来怎样、具体结果。
3. 给出来源链接。
4. 给出 AI建议星级 和 AI建议理由。
5. 不要填写正式星级，正式星级由我在飞书确认。
```
